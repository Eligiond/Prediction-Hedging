RISKOFF FOR MACOS

1. Keep Install Riskoff.command and riskoff-source.tar.gz together.
2. Double-click Install Riskoff.command.
3. Approve the file if macOS asks.
4. Keep the Terminal window open.
5. Paste the copied HTTPS link into Claude Settings > Connectors.

Requires Node.js 20+ and Python 3.9+.
Riskoff is paper trading software. It cannot place real-money orders.
